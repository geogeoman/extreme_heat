    TensorFlow PyTorch NLP

  • 冯同学

    可视化工程师

    负责爬虫开发和多媒体内容收集，专注于数据可视化和用户体验设计。 在网页开发和交互设计方面有丰富经验。

    JavaScript D3.js 爬虫

  • 沈同学

    前端开发

    负责网站开发和用户界面设计，专注于响应式布局和交互体验优化。 在现代Web技术方面有深入研究和实践经验。

    React Vue UI设计

  • 胡同学

    视频编辑 AI生成 抖音

团队协作

我们采用现代化的协作方式，确保项目高效推进和成果质量保证

分工与合作

明确分工

每位团队成员都有明确的职责分工，同时保持密切的协作沟通， 确保各个环节的无缝衔接。
