# 【Web 前端实战】基于 ECharts 与 Anime.js 的极端高温 AI 防灾可视化交互项目（含自研图表组件）

## 项目简介
本项目是一个针对极端气候研究的前端可视化实战案例。项目重点展示了如何基于 **ECharts** 封装具有复杂业务逻辑（多源数据对比、自动平均线计算、动态警戒线）的图表组件，并结合 **p5.js** 与 **Anime.js** 实现了高性能的沉浸式背景动效。

项目不仅包含完整的响应式 UI 设计，还提供了企业级目录结构和组件化的代码实现，适合用于学习前端数据可视化和交互设计。

## 核心亮点
* **自研图表组件 (`temperature-chart.js`)**：
    * 支持动态传入不同城市的温度数据。
    * 自动计算多数据源平均趋势线。
    * 内置巴黎协定 1.5°C 警戒线标注。
    * 提供完整的 **TypeScript 类型定义** (`.d.ts`)，开发体验更佳。
* **沉浸式交互**：
    * 基于 p5.js 的热浪粒子背景动画。
    * Anime.js 驱动的卡片交错入场动效。
